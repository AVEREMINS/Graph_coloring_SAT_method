# Раскраска графа через SAT

## Что надо

Есть граф и надо покрасить вершины.
Главное условие простое: если две вершины соединены ребром, то цвета у них должны быть разные.

В лоб это перебирать долго, поэтому задача переводится в SAT.

## Идея

Для каждой вершины и каждого цвета делается булевая переменная.

Дальше добавляются условия:

1) у каждой вершины есть какой-то цвет

2) у одной вершины не может быть сразу два цвета

3) две соседние вершины не могут иметь один цвет

После этого получается обычная КНФ.
Её уже можно отдавать SAT solver.

Если граф большой, то сначала берётся жадная раскраска, а потом SAT пробует улучшить ответ.
Так работает быстрее и не умирает на больших тестах.

## Что лежит в файлах

`Graph.h` - граф

`Answer.h` - ответ

`InputReader` - чтение графа

`OutputWriter` - вывод ответа

`GreedyColor` - жадная раскраска

`SatFormula` - построение КНФ

`SatSolver` - свой SAT solver

`ColorSatSolver` - подбор раскраски через SAT

`SatPlus` - порядок вершин и random jump

`CheckColor` - проверка ответа

`main.cpp` - всё связывает

## По результатам

На маленьких графах SAT нормально подбирает цвета.
На больших графах прямой SAT становится тяжёлым, поэтому там помогает жадная раскраска.

В таблице отдельно записаны размеры графа, сколько цветов получилось и время.
